# doronuma_game_spec.md
英名: doronuma_sabotage
和名: 泥沼の妨害

## 1. ゲーム概要
* **コンセプト:** お互いに罵り合いながら邪魔をし合うゲーム。妨害が中心で、全員がマイナス点で終わることがほとんど。将来的に紙製のボードゲームとしても遊べる設計。
* **プレイ人数:** 3〜5人
* **勝利条件:** ゲーム終了時に「最も合計点数が高い（マイナスが少ない）」人の勝利。
    * **同点の場合:** 順位は同じとする。
    * *順位の例:* 1位：Aさん（3点）、1位：Bさん（3点）、3位：Cさん（-1点）、4位：Dさん（-3点）

## 2. 準備
* **初期点数:** 全員「0点」からスタートする。
* **勝利点カード（山札）:** 引くたびに以下の確率で点数がランダムに決定される（紙製の場合はこの確率に基づいた枚数のカードを用意する）。
    * マイナス（-1点〜-5点）：約80%
    * ゼロ（0点）：約15%
    * プラス（+2点〜+5点）：約5%
* **行動カード（山札）:** 全50枚。山札を準備する際、下半分（25枚の中）にのみ「突然死カード」を1枚混ぜる。
* **初期手札（行動カード）:** 手札の上限は5枚。順番による有利不利をなくすため、最初の枚数を変える。
    * 1番目の人：3枚
    * 2番目の人：4枚
    * 3番目以降の人：5枚

## 3. ターンの進行
時計回りに順番に行動する。自分の番が来たら、以下の4つのうちから1つを選んで実行し、ターンを終了する。
手札が上限の5枚を超える場合は、5枚になるように捨てる。カードを引くなどして手札が上限を超えた場合、超過した分を捨てるカードを自分で選択するフェーズに入る。この選択中に制限時間切れになった場合は、追加でカードを引くことはせず、現在の手札からランダムに上限（5枚）になるように自動で捨てられる。実行できない行動は選択できない。

1.  **2枚引いて終了:** カードを2枚引く（カードは使えない）。
2.  **1枚引いて1枚使う:** カードを1枚引き、手札からカードを1枚使う。
3.  **行動カードを1枚捨てて2枚まで使う:** ドローせず、手札のカードを1枚捨てることで、手札からカードを2枚使う。
4.  **パス:** 何も行動しない。山札がなくなった際などに選択する。

## 4. 割り込み処理
相手の行動に対するカウンターとして、自分の番以外でも割り込んでカードを使用できる。

* **処理の順番:** 誰かが割り込みカードを使うたびに確認時間を挟み、最終的に一番最後に使われたカードから逆順に（後出し有利で）効果を処理する。
* **対象の明確化:**
    * 全員を対象とする攻撃（例：道連れ）は、各プレイヤーへの個別の攻撃が集まったものとして扱う。
    * 「完全無効」などの防御カードは、それを使った本人への攻撃のみを防ぐ。
    * 「強奪」や「出る杭を打つ」などの効果に対して「完全無効」が使われた場合、その行動そのものがキャンセルされる。

## 5. 終了条件とペナルティ
* **終了トリガー:** 誰かが山札から「突然死」を引いた瞬間に終了へのカウントダウンが始まる。
* **ペナルティ:** 「突然死」を引いたプレイヤーは、即座に勝利点カードを「3枚」引き、自分の点数に加算する。
* **ラストターン保証:** 「突然死」が出ても即座にゲームを終わらせず、「その1周が終わるまで（一番最後に行動する人の番が終わるまで）」ゲームを続ける。全員の行動回数が同じになった時点で終了し、最終的な点数計算を行う。

## 6. 脱落・放置への対応（ルール上の扱い）
* 連続して行動できないプレイヤーは「空席（不参加）」扱いとなる。ゲームからは外さないが、最終点数の計算時に強制的に最下位（負け）となる。
* 空席のプレイヤーの順番が来たら、自動的に「2枚引き、上限に合わせてランダムに捨てる」処理が即座に行われる。空席のプレイヤーを攻撃の対象にすることは可能。

## 7. 行動カード一覧
* **単なる嫌がらせ (15枚):** 指定した1人に勝利点カードを1枚引かせる。
* **道連れ (8枚):** 自分以外の全員に勝利点カードを1枚ずつ引かせる。
* **集中砲火 (4枚):** 指定した1人に勝利点カードを2枚引かせる。
* **完全無効 (7枚):** 相手が使った行動カードの効果を完全に打ち消す。自分への攻撃のみを無効化し、強奪などに使った場合はその行動自体をキャンセルする。
* **なすりつけ (6枚):** 自分への行動カードの効果を、指定した別の人にそらす。
* **倍返し (4枚):** 勝利点カードを引かされそうになった時、それを無効にして攻撃者に2枚引かせる。
* **強奪 (3枚):** 相手1人を指定し、その人が持っている「プラスの勝利点カード」を1枚奪う。
* **出る杭を打つ (2枚):** 現在プラス点になっているプレイヤー全員の、プラスの勝利点カードをすべて捨てさせる。
* **突然死 (1枚):** 引いた瞬間にペナルティ発動。引いた本人が勝利点カードを3枚引き、そのラウンドの最後の手番の人が終わるとゲーム終了。
