# doronuma_game_screen.md

## 1. ゲーム画面の構成（合成フェーズ中）

自分のターン開始時、まず合成フェーズに入ります。

```
-----------------------------------------------------------
【中央・左側：プレイエリア】         | 【右側：ゲーム情報】
                             |
[山札: 残り35枚]   [捨て札]     | [ 自身のターン時間タイマー: 15秒 ]
                             |
    ┌────────────┐           | ▼ プレイヤー情報
    │            │           | ・[手番] Aさん: 3点 (手札4枚)
    │  合成フェーズ │           | ・Bさん: -2点 (手札5枚)
    │            │           | ・Cさん: 0点 (空席 / 攻撃対象外)
    └────────────┘           |
                             | ▼ 行動ログ
                             | ・Aのターンが始まりました
                             | ・+1点カードを受け取りました
                             |
-----------------------------------------------------------
【下部：あなた（プレイヤー）のエリア】

  ✨ +1点カードを受け取りました！

  【手持ちの勝利点カード】 +1 × 3枚  /  +3 × 1枚  /  +5 × 0枚  /  -3 × 1枚

  [ 昇格（小）: +1×2 → +3 ] [ 昇格（大）: +3×2 + -3×1 → +5 ] [ 合成しない ]
  (※ 素材が足りないボタンは暗く表示され、押せない状態になる)
-----------------------------------------------------------
```


## 2. ゲーム画面の構成（ドロー＆アクションフェーズ・通常時）

合成フェーズが終わると、ドロー＆アクションフェーズに移行します。

```
-----------------------------------------------------------
【中央・左側：プレイエリア】         | 【右側：ゲーム情報】
                             |
[山札: 残り35枚]   [捨て札]     | [ 自身のターン時間タイマー: 12秒 ]
                             |
    ┌────────────┐           | ▼ プレイヤー情報
    │            │           | ・[手番] Aさん: 3点 (手札4枚)
    │ 現在実行中の │           | ・Bさん: -2点 (手札5枚)
    │   カード   │           | ・Cさん: 0点 (空席 / 攻撃対象外)
    │            │           |
    └────────────┘           | ▼ 行動ログ
                             | ・Aが「道連れ」を使用！
                             | ・Bが「完全無効」で割り込み！
                             |
-----------------------------------------------------------
【下部：あなた（プレイヤー）のエリア】

  [2枚引いて終了] [1枚引いて1枚使う] [引かずに捨てて2枚まで使う] [パス]
  (※ 手札の枚数条件を満たさないボタンは暗く表示され、押せない状態になる)

  [カードA] [カードB] [カードC] [カードD] [カードE]
-----------------------------------------------------------
```


## 3. ゲーム画面の構成（相手を選ぶ時の状態）

「強奪」など、特定の相手を選ぶカードを使った直後の画面の変化です。

```
-----------------------------------------------------------
【中央・左側：プレイエリア】         | 【右側：ゲーム情報】
                             |
[山札: 残り35枚]   [捨て札]     | [ 自身のターン時間タイマー: 10秒 ]
                             |
    ┌────────────┐           | ▼ プレイヤー情報（ここから選ぶ）
    │            │           | ・[手番] Aさん: 3点 (手札4枚)
    │  強奪(選択中) │           | ・[ 決定 ] Bさん: -2点 (手札5枚)
    │            │           | ・Cさん: 0点 (空席 / 攻撃対象外)
    └────────────┘           |   (※ 空席は「決定」ボタンが表示されない)
                             | ▼ 行動ログ
                             | ・Aの順番が始まりました
                             |
-----------------------------------------------------------
【下部：あなた（プレイヤー）のエリア】

  [ 右側のリストから、攻撃する相手を選んでください ]  [ 選ぶのをやめる ]

  [カードA] [カードB] [カードC] [カードD] [カードE]
  (※ カードは暗く表示され、選べない状態になる)
-----------------------------------------------------------
```


## 4. 各画面部品の役割と動き

* **自身のターン時間タイマー:**
    * 画面右上の最も目立つ場所に配置します。
    * 自分のターン開始（合成フェーズ含む）から、ドロー＆アクションフェーズの終了までカウントダウンが動きます。
    * 割り込みの受付時間にも別途カウントダウンが動きます。
    * ゼロになると**パスが自動的に選択されます**（手札に変化はありません）。

* **プレイヤー情報:**
    * 全員の現在の合計得点（手持ち勝利点カードの合計値）、行動カードの手札枚数、状態（空席など）を一覧で表示します。
    * 現在誰の番かわかるように「手番」のマークをつけます。
    * 相手の行動カードの手札枚数を表示することで、割り込みされるかどうかの予測を立てやすくします。
    * **空席（AFK）プレイヤーには「攻撃対象外」の表示を行い、対象選択時に「決定」ボタンを表示しません。**

* **合成フェーズのUI:**
    * 自分のターン開始時に「+1点カードを受け取りました！」の通知と現在の手持ち勝利点カードの内訳を表示します。
    * 「昇格（小）」「昇格（大）」の合成ボタンを表示します。素材が足りない場合はボタンを暗く（disabled）表示します。
    * 「合成しない」ボタンを押すか、合成が完了すると、自動的にドロー＆アクションフェーズへ移行します。

* **アクションボタン:**
    * 手札の枚数条件を満たさない選択肢はボタンを暗く（disabled）表示し、押せない状態にします。
        * 「1枚引いて1枚使う」→ 引く前の手札が5枚の場合は選択不可
        * 「引かずに捨てて2枚まで使う」→ 捨てる前の手札が1枚以下の場合は選択不可

* **相手を選ぶ仕組み（対象選択）:**
    * 相手を選ぶカードを使った場合、下部の自分の操作エリアが「案内メッセージ」と「選ぶのをやめる」ボタンに切り替わります。
    * 同時に、右側の「プレイヤー情報」の通常プレイヤーの名前の横に、相手を選ぶための「決定」ボタンが表示されます。
    * 空席（AFK）プレイヤーには「決定」ボタンを表示しません（攻撃対象にできないため）。
    * 相手を選んでいる間は、間違えて別の手札を使わないように、手札は暗く表示されて押せない状態になります。

* **割り込みオーバーレイ:**
    * **妨害系カード**（単なる嫌がらせ・道連れ・集中砲火・泥沼引き込み）が使われた時のみ、割り込みの受付オーバーレイが表示されます。
    * 得点取得系・強奪系カードが使われた場合は割り込みオーバーレイは表示されません。
    * オーバーレイには「妨害への対抗」カテゴリのカード（完全無効・なすりつけ・倍返し・はじき返し）のみが選択肢として表示されます。

* **捨て札:**
    * 「引かずに捨てて2枚まで使う」行動で手放したカードを置く場所として、山札の横に配置します。
